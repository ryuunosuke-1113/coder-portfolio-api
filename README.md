# API投稿一覧 + 投稿追加（JavaScript）

JavaScriptでAPIから投稿データを取得し、一覧表示するアプリです。  
ローカルで投稿を追加でき、検索・表示件数切替・再読み込みにも対応しています。

## Features
- API取得（fetch / async-await）
- ローディング表示 / エラー表示
- 投稿一覧表示（API + ローカル追加）
- 検索（タイトル部分一致）
- 表示件数切替（5 / 10 / 20）
- 再読み込み

## Tech
- HTML / CSS
- JavaScript（Vanilla JS）
- API：JSONPlaceholder

## How to Run
1. このリポジトリをクローン
2. `index.html` をブラウザで開く

## Notes
- 投稿追加はローカル配列に保存されるため、ページ更新でリセットされます。
